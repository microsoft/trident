# Requirements

Trident's build-time dependencies.

## Rust

- Rust (latest stable)

## Packages

- Ubuntu/Debian:

  ```bash
  sudo apt install build-essential pkg-config libssl-dev libclang-dev protobuf-compiler
  ```

- Python

  ```bash
  sudo pip3 install virt-firmware
  ```
